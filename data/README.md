# Data Folder

This project uses the built-in `digits` dataset from `sklearn.datasets`.

No manual dataset download is required.

The dataset is loaded directly in `utils.py` using:

```python
from sklearn.datasets import load_digits
```

Why this folder exists:

- to keep the project structure clean
- to show where dataset notes or future download scripts would go
- to match the internship project requirement for a `data/` folder
